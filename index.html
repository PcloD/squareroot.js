Squareroot.js

Squareroot.js was created by Bartek Drozdz and is a fruit of many years of experience in building cross-device WebGL projects.

Squareroot is a concise inteface over the WebGL API. The code evolved from experiences on different production WebGL projects. The goal was for SQR to help writing creative with code rather than being easy to use. It is composed mostly of tools that will make creating WebGL experiences easier. The code originates from a practial need encontered during one project or another, not from a priori assumptions of how a 3d engine should work. A lot of the structure is inspired by the Unity3d API, including the central concept of 'transforms' as building blocks.

The first version of this engine was created 6 years ago - it was called J3D back then. After that, two versions of Squareroot.js followed. Now in Version 3, each consecutive version was hand crafted from scratch; carefully discarding the inefficeint in favor of speed and power.
