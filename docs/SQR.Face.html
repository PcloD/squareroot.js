Class: Face

SQR. Face

Constructor

new Face()

Face is a triangle or a quad. If the face is a quad, both triangles composin the quad, shader the same normal - thanks to this flat shaded materials have quads shaded the same way which is nicer than havong each triangle have a slightly different normal. Currently it supports the following attributes: aPosition, aNormal, aUV.

Calculte the normal for this face. Regardless of whether there are 3 or 4 vertices the normal is calculated for the frst 3 of them an applied to the entire face.
